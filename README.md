A feature-rich calculator developed to strengthen **JavaScript logic, DOM manipulation, and UI interaction skills**.  
It includes both **basic and scientific operations**, along with a visually engaging interface.

---

## 🎯 Purpose of This Project

This project was created to improve my understanding of:

- JavaScript class-based architecture  
- DOM manipulation & event-driven programming  
- Managing application state (operands & operations)  
- Handling edge cases in calculations  
- Building interactive UI without using frameworks  

Although it started as a practice project, it evolved into a **fully functional scientific calculator with advanced UI effects**.

---

## 🚀 Features

### 🔢 Core Functionalities
- Basic operations: `+`, `−`, `×`, `÷`
- Decimal support
- Delete (DEL) & All Clear (AC)
- Keyboard input support
- Error handling (e.g., division by zero)

### 🧪 Scientific Mode
- Trigonometric functions: `sin`, `cos`, `tan`
- Logarithmic functions: `log`, `ln`
- Square root (√)

---

## ⚙️ How It Works

- The calculator logic is implemented using a **JavaScript class**
- Maintains:
  - `currentOperand`
  - `previousOperand`
  - `operation`
- Uses event listeners for:
  - Button interactions  
  - Keyboard input  
  - Mode switching  
- Performs real-time computation and display updates
